## 🎯 install local development:

```zsh
> git clone this repo
> install dependencies `pip install -r requirements.txt`
```

```zsh
> install venv `python -m venv env`
> run command in linux or mac `. env/Scripts/activate` in windows `env/Scripts/activate.bat`

```
